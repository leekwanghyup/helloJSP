## JSP 내장 객체 
- request : 사용자 요청 처리 
- response : 서버 응답처리 
- out : 웹브라우저 출력 
- session : 사용자 인증 정보 관리 
- application : 서버 정보 관리 
- exception : 예외처리 
- config : jsp 환경 정보 처리 
- page : 현재 페이지 정보 처리 

## 변수 사용범위 
- pageContext : 현재페이지
- reqeust : 요청,응답 페이지
- session : 각가의 사용자 마다 변수 할당 
- application : 서버 변수(애플리케이션 전체에서 사용 가능함)
